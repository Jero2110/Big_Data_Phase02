
## Para el archivo *docker-compose.yml*
En la terminal se pueden ingresar las siguientes líneas de código:
- Iniciando la imagen definida en el archivo:
docker compose build

- Para cargar e inicializar el docker:
docker compose up

- Para detener el docker y reiniciar:
Ctrl + C
docker compose up --build

- Para destruir el docker:
docker compose down -v
